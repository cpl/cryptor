# Cryptor Docs

This branch (`gh-pages`) contains the GitHub Pages website with all of Cryptor's documentation.

## Quick Introduction

* The theme/template used is [pmarsceill/just-the-docs](https://github.com/pmarsceill/just-the-docs)
* Documentation contributions must be made to the `/docs` directory
* Documentation files must be Markdown
* Any contribution is welcome
* For any other information see
  * [Jekyll Docs](https://jekyllrb.com/docs/)
  * [just-the-docs live demo](https://pmarsceill.github.io/just-the-docs/)
